Crée une application SaaS full-stack moderne pour la gestion d’entreprise en Mauritanie, avec :

- Frontend : React + Vite
- Backend : Node.js + Express
- Base de données : PostgreSQL
- ORM : Prisma
- Authentification : Replit Auth pour la connexion, puis gestion des rôles et permissions dans l’application
- UI propre, professionnelle, responsive, orientée back-office
- Architecture multi-tenant : chaque entreprise doit avoir ses propres données via company_id

Objectif produit :
Développer deux modules principaux :
1. Facturation
2. GRH (Ressources Humaines)

L’application doit être prête à évoluer ensuite vers une suite ERP légère.

====================================
1. EXIGENCES GÉNÉRALES
====================================

Construis une application structurée, sécurisée et maintenable avec :
- séparation claire frontend/backend
- API REST
- validations backend
- gestion des erreurs
- logs propres
- seed initial pour rôles, permissions et compte administrateur de démonstration
- code propre, commenté, cohérent

Le projet doit inclure :
- tableau de bord
- authentification

- gestion des utilisateurs
- gestion des rôles et droits d’accès
- module Facturation
- module GRH

====================================
2. MULTI-TENANT
====================================

L’application doit fonctionner pour plusieurs entreprises.

Toutes les données métier doivent être rattachées à une entreprise avec :
- company_id
- created_by
- updated_by
- created_at
- updated_at

Un utilisateur ne peut voir que les données de son entreprise, sauf Super Admin plateforme.

Utiliser deux niveaux de sécurité :
1. Authentification utilisateur
2. Autorisation fine par rôle et permission

====================================
4. RÔLES À CRÉER
====================================

Créer les rôles suivants :
- Super Admin
- Admin Entreprise
- Comptable
- RH
- Manager
- Employé

Définition :
- Super Admin : accès global à toutes les entreprises, utilisateurs, modules et paramètres
- Admin Entreprise : accès complet à son entreprise uniquement
- Comptable : accès complet au module facturation, lecture partielle RH si nécessaire
- RH : accès complet au module RH, lecture limitée facturation
- Manager : accès lecture tableaux de bord, employés de son périmètre, validation de certaines demandes
- Employé : accès restreint à son propre profil, ses congés, ses documents personnels
